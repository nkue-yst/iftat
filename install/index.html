Install

Homebrew⌗

iftat supports installation via Homebrew.

$ brew tap nkue-yst/brew
$ brew install iftat

Install yourself⌗

Clone the project from GitHub and compile it like follows.

$ git clone https://github.com/nkue-yst/iftat.git
$ cd iftat
$ cargo build --release

Then, cargo build the resultant iftat executable on target/release directory.
